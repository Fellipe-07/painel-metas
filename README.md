# painel-metas
Metas Diarias

O Painel de Metas Diárias é uma plataforma criada para ajudar você a organizar seu dia com foco, produtividade e clareza. 
Aqui, você pode definir objetivos, acompanhar suas metas e visualizar suas conquistas de forma simples e intuitiva. Além de facilitar sua rotina para pessoas com a rotina apertada e acaba não se lembrando de suas metas Diárias ou prolongadas, definindo a prioridade, data e adicionar descrição.

## Funcionalidades

- Adicionar Descrição
- Adicionar Prioridade
- Remover Meta

## Como Rodar

        1. Baixe o repositório (Clique no Botão “Code” > Download ZIP)
		
        2. Extraia a pasta
		
        3. Abra o arquivo index.html no navegador

## Como Clonar

No terminal do VS Code digite:

	git clone https://github.com/Fellipe-07/painel-metas.git
	
No GitHub Desktop:

	No menu: File(Arquivo) > Clone a repositor(Clonar repositório) > Selecione a aba URL, no campo URL, cole o link do repositório https://github.com/Fellipe-07/painel-metas.git
